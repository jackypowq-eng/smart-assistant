# Smart Assistant
一个轻量、易用的智能助手工具，支持对话交互、任务处理、自动化操作等功能。

## 功能特性
- 智能对话交互
- 自定义指令与快捷操作
- 支持多场景使用
- 轻量部署，开箱即用
